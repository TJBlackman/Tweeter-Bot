What are API keys?

API keys allows us to communicate directly with Twitter on your behalf! You can think of them as highly complex passwords that we present to Twitter to prove that we have the authority to post, like, and retweet from your account.

Without them, we simply cannot access your account.

Note: Providing API keys from Twitter will not give us access to see your Twitter password, or other personal information.

How do I generate my API keys?
  • Log in to Twitter, then visit https://apps.twitter.com/app/new
  • Fill in the required fields to create a new Twitter application
    • Name: Twitter Bot
    • Description: Automated Twitter Service
    • Website: https://www.twitterbot.io
  • Submit application
  • Near the top of the page, select Keys & Access Tokens
  • Copy and paste API Key and API Secret
  • Click Create Access Token
  • Copy and paste Access Token & Token Secret
  • Save settings on TwitterBot

You may revoke TwitterBot access to your account at any time by logging in via the link above and clicking the revoke access button at the bottom of the API Keys page.
